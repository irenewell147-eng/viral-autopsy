# viral-autopsy · 再拆N遍

> Paste any viral post. Get the breakdown + your own rewrite. No formulas — just the logic behind why it works.
> 
> 不卖公式，只教你看懂。把爆款文案粘进来，一次性给你拆解报告+改写版本。

---

## What it does · 能做什么

- **Decode viral content · 拆解爆款**：找到它命中了谁、用了什么钩子、底层逻辑是什么
- **One input, two outputs · 一次输入，两部分输出**：拆解报告 + 改写版本，不需要多轮对话
- **Never goes stale · 永不过时**：不依赖固定公式，教你看懂判断逻辑，每周新爆款都能用

---

## How to use · 怎么用

**Step 1 · 第一步：加载SKILL.md**

支持以下工具 / Works with:
- Claude / ChatGPT / DeepSeek（粘贴SKILL.md全文作为系统提示词 / paste as system prompt）
- Cherry Studio（新建助手，粘贴为系统提示词 / new assistant, paste as system prompt）
- 支持SKILL.md格式的Agent工具 / Any Agent tool that supports SKILL.md format

**Step 2 · 第二步：按格式输入**

```
【爆款文案】：（粘贴你找到的爆款原文 / paste the viral content here）
【我的账号定位】：（你做什么内容、面向谁 / your account niche and target audience）
```

**Step 3 · 第三步：直接用输出结果**

你会得到 / You'll get:
1. 拆解报告（为什么火）/ Breakdown（why it works）
2. 改写版本（适配你的定位）/ Rewrite（adapted to your niche）
3. 备选方向（换个角度的开头）/ Alternative angle（different hook to try）

## Example · 使用示例

**Input · 输入：**
```
【爆款文案】：容貌焦虑个屁呀，高P发朋友圈又怎么样？
现在高P已经算很真实了好不好？
和AI换脸比起来，显得我们这种高P手艺人真的很傻。

【我的账号定位】：AI自媒体博主，教普通人用AI做小红书内容
```

**Output · 输出示例：**

拆解 → 命中有容貌焦虑的年轻人，反常识暴论+降维打击+身份重建，帮观众骂世界顺便送新身份

改写版本 →
> 内容焦虑个屁呀，肝到凌晨写文案又怎么样？
> 现在认真做内容已经算很良心了好不好？
> 和那些AI一键洗稿批量发的比起来，
> 显得我们这种手工创作者，真的很傻。
