# 再拆N遍 · SKILL

> 不卖公式，只教你看懂。

把任意一条爆款文案粘进来，告诉我你的账号定位——
我帮你拆清楚它为什么火，然后直接改写成你自己的版本。

---

## 这个Skill能做什么

- **拆解爆款**：命中了谁、用了什么钩子、底层逻辑是什么
- **一次输出**：拆解报告 + 改写版本 + 备选方向，不需要多轮对话
- **永不过时**：不依赖固定公式，教你看懂判断逻辑，每周新爆款都能用

---

## 怎么用

**第一步：把SKILL.md加载到你的AI工具里**

支持以下工具：
- Claude / ChatGPT / DeepSeek 等对话框（直接粘贴SKILL.md全文作为系统提示词）
- Cherry Studio（新建助手，粘贴为系统提示词）
- 支持SKILL.md格式的Agent工具（直接加载文件）

**第二步：按格式输入**

```
【爆款文案】：（粘贴你找到的爆款原文）
【我的账号定位】：（一两句话：你做什么内容、面向谁）
```

**第三步：直接用输出结果**

AI会一次性给你：
1. 拆解报告（为什么火）
2. 改写版本（适配你的定位，可以直接录）
3. 备选方向（换个角度的开头思路）

---

## 使用示例

**输入：**
```
【爆款文案】：容貌焦虑个屁呀，高P发朋友圈又怎么样？
现在高P已经算很真实了好不好？
和AI换脸比起来，显得我们这种高P手艺人真的很傻。

【我的账号定位】：AI自媒体博主，教普通人用AI做小红书内容
```

**输出示例：**

拆解报告 → 命中有容貌焦虑的年轻人，用反常识暴论+降维打击+身份重建完成共鸣

改写版本 → 
> 内容焦虑个屁呀，肝到凌晨写文案又怎么样？
> 现在认真做内容已经算很良心了好不好？
> 和那些AI一键洗稿批量发的比起来，
> 显得我们这种手工创作者，真的很傻。
